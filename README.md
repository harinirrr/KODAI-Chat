# KodAI Chat App

This is the fullstack KodAI Chat app.

- **Frontend:** React + Vite chat UI
- **Backend:** Express server calling Hugging Face Space
- Supports private Spaces via HF_TOKEN environment variable

## Setup

1. Clone the repo
2. Install dependencies (`npm install`) in both `frontend` and `backend`
3. Add `.env` locally if Space is private
4. Run frontend (`npm run dev`) and backend (`node index.js`)
